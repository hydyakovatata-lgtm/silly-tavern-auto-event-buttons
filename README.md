Оставь надежду сюда входящий

Установка: https://github.com/Delidgi-meow/silly-tavern-auto-event-buttons

### Описания кнопок:

- **СЮЖЕТНЫЙ ПОВОРОТ** (<i class="fa-solid fa-bolt"></i>): Вводит случайный неожиданный поворот сюжета, такой как новый NPC, поломка, секрет или изменение окружения.
- **ХАОС И ДРАМА** (<i class="fa-solid fa-fire"></i>): Добавляет непредсказуемое событие, приводящее к полному хаосу и драме в истории.
- **НЕЙТРАЛЬНЫЙ** (<i class="fa-solid fa-arrows-rotate"></i>): Продолжает сцену нейтрально, возвращая неразрешённый сюжетный момент из прошлого.
- **РАНДОМ** (<i class="fa-solid fa-dice"></i>): Вводит случайный элемент из большого пула идей (сюжетный твист, персонаж, катастрофа и т.д.), оставаясь в рамках сеттинга.
- **ПОЛНЫЙ ХАОС** (<i class="fa-solid fa-skull"></i>): Вводит случайный элемент из большого пула, но с акцентом на креативность и дикость, игнорируя окружение.

### Инструкция по использованию:
1. Панель "Быстрые События" появляется в правом верхнем углу (перетаскивается мышкой).
2. Нажмите любую кнопку — промпт добавится скрыто в чат.
3. Отправьте сообщение в чат — AI применит поворот в ответе.
4. Промпт очищается автоматически после ответа AI.
5. На мобильных: Используйте мини-кнопку для показа/скрытия панели.
6. Уведомление покажет тип применённого поворота.
7. Увидеть полное описание применяемого промпта - велком ту консоль.
